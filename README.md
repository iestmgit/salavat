# 📿 Digital Tasbih - تسبیح دیجیتال

A professional digital tasbih (prayer counter) with auto-save, 100+ languages, and smart goal tracking.

## 🌟 Features
- 🔄 Auto-save to localStorage
- 🌍 100+ languages with RTL/LTR support
- 🎯 Smart goal setting & progress tracking
- 💾 Data recovery on page load
- 📁 Custom logo support (1.png)
- ⌨️ Keyboard shortcuts
- 📱 Fully responsive
- 🎨 Glassmorphism UI design

## 🏢 Brand
**Produced by iestm company**

## 🚀 Quick Start
1. Upload all files to your server
2. Place `1.png` for custom logo (optional)
3. Open `index.html` in browser
4. Start counting!

## ⌨️ Shortcuts
- **Space**: Count +1
- **Backspace**: Undo
- **R**: Reset

## 🌍 Supported Languages
100+ languages including: فارسی, English, العربية, Türkçe, اردو, हिन्दी, 中文, 日本語, 한국어, Русский, Français, Español, Deutsch, Italiano, Português, and many more!
